parse_input_records(fasta_file: Optional[str] = None, out_dir: str = '', genome_size_cutoff: float = False, write_single: bool = False, plot: bool = True, genome_cutoff_type: Optional[float] = None, platform: str = 'SLK', preset_type: Optional[str] = None, deduplicate: bool = False)¶

This function parses an input multi-record fasta file containing the genomes of the target pathogen. If genome_size_cutoff is desired, then the float is a fraction of the largest record length in the set. If write_single is desired, then multi_record files are converted into single entry fasta files.

Parameters

  • fasta_file (str, optional) – path to fasta, defaults to None

  • out_dir (str, optional) – directory to print out to, defaults to ‘’

  • genome_size_cutoff (float, optional) – percent cutoff, defaults to False

  • write_single (bool, optional) – write files to single fasta, defaults to False

  • plot (bool, optional) – plot curation analysis, defaults to True

  • genome_cutoff_type (float, optional) – max or median cutoff thresholding, defaults to None

  • platform (str, optional) – SLK or ISR, defaults to ‘SLK’

  • preset_type (str, optional) – Small Medium or Large, defaults to None

  • deduplicate (bool, optional) – remove duplicate sequences except first occurance, defaults to False
